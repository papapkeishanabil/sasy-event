Navigasi ke Realtime Settings

Lokasi Realtime di Supabase Dashboard

📍 Cara 1: Lewat Database

1
Klik "Database" di sidebar
Di sidebar kiri, cari dan klik menu Database
2
Cari tab "Replication" di atas
Di halaman Database, ada tabs di atas. Klik tab Replication
3
Pastikan "Realtime" toggle ON
Di section Realtime replication, pastikan toggle Enable Realtime dalam posisi ON (hijau)
▸ Sidebar → Database
▸ Tab atas → Replication
▸ Section → Realtime → Toggle ON

📍 Cara 2: Lewat Project Settings

1
Klik logo project di atas kiri
"Sasienala" → pilih Project Settings
2
Pilih "Database" di settings
Di menu settings kiri, klik Database
3
Cari "Realtime" section
Scroll ke bawah sampai menemukan Realtime settings
▸ Klik logo Sasienala → Project Settings
▸ Menu kiri → Database
▸ Scroll → Section Realtime
🚀 Buka Database → Replication (Langsung) ⚙️ Buka Project Settings → Database ↗️ Buka Aplikasi untuk Test
