Cara Akses Aplikasi dari HP:

CARA 1 - Local Network (WiFi sama):
1. Pastikan HP dan komputer di WiFi yang SAMA
2. Buka browser HP, ketik: http://192.168.1.15:8080/

CARA 2 - Direct File (Kalau WiFi beda):
1. Buka folder: d:\PROJECTS\SASIE REGIST\dist
2. Kirim file "index.html" ke HP via:
   - WhatsApp / Email / Google Drive
3. Buka file tersebut di browser HP
4. Aplikasi akan jalan!

CARA 3 - Hotspot (Kalau tidak ada WiFi):
1. Nyalakan Hotspot di HP
2. Connect komputer ke hotspot HP
3. Buka di HP: http://192.168.1.15:8080/

Info:
- IP Komputer: 192.168.1.15
- Server Port: 8080
- Status: Running
